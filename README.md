# MarketMate – Sentiment Analysis Web Application

A full-stack web application that classifies customer review 
sentiments (positive / negative / neutral) using Machine Learning 
and EDA, with interactive dashboards for business insights.

## Tech Stack
- Python, Flask
- Pandas, NumPy, Scikit-learn
- Matplotlib / Power BI
- HTML, CSS, JavaScript

## Features
- Sentiment classification of customer reviews
- EDA pipeline for data cleaning and preprocessing
- Interactive visualization dashboards

## How to Run
pip install -r requirements.txt
python app.py
